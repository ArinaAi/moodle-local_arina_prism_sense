# CHANGES

## 1.5 (2026-03-25)

Initial public release.

### Features
- AI-powered content generation: converts course PDFs into structured Moodle lectures via the Arina AI service.
- Generated content types: slides, table-of-contents, quizzes, and rich HTML lecture pages.
- Content approval workflow: teachers generate content, managers/admins approve and publish to the course.
- Student completion tracking: tracks which students have viewed each AI-generated content item.
- Credit management: integrated with the Arina credit wallet; usage dashboard available in the CMS view.
- IOMAD multi-tenancy support: per-company API keys and isolated credit pools.
- Content export: download generated content as PPTX or PDF.
- Privacy API: full implementation of Moodle's Privacy API for data export and deletion.
- Azure Blob Storage: optional support for storing PDF source files in Azure Blob Storage.
